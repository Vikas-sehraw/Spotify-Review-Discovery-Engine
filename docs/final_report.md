# AI-Powered Review Discovery Engine

## Executive Summary

This project analyzes Spotify user feedback from multiple public sources using an AI-powered review classification workflow.

The objective is to identify barriers to meaningful music discovery and recommend product improvements that reduce repetitive listening behaviour.

---

# Data Sources

- Google Play Reviews
- Apple App Store Reviews
- Reddit Discussions
- Spotify Community
- Social Media

---

# AI Workflow

1. Collect reviews
2. Merge into a single dataset
3. AI classifies every review
4. Generate structured insights
5. Build dashboard
6. Recommend product improvements

---


# 1. Why do users struggle to discover new music?

Major themes include:

- Recommendations frequently surface familiar artists.
- Users rely heavily on saved playlists.
- Discovery playlists become repetitive.
- Recommendation diversity is limited.
- Search behaviour favours known songs.

# 2. Most common frustrations

- Repetitive recommendations
- Playlist repetition
- Limited recommendation diversity
- Difficulty finding emerging artists
- Weak personalization for discovery

# 3. Listening behaviours

Users typically listen for:

- Familiar music
- Workout sessions
- Relaxation
- Studying
- Mood-based playlists
- Artist exploration

# 4. Why users repeatedly listen to the same music

Reasons include:

- Existing playlists are convenient.
- Recommendation algorithms prioritize familiarity.
- Users trust familiar music more than unknown artists.
- Limited exploration incentives.

# 5. User segments facing discovery challenges

Most affected:

- Casual listeners
- Premium subscribers
- Long-term Spotify users

Least affected:

- Users who actively search for new music.

# 6. Unmet needs

Users consistently express interest in:

- More diverse recommendations
- Better genre exploration
- Greater control over recommendation freshness
- Discovery of smaller artists
- Transparency into why songs are recommended

# Proposed Feature

Spotify Discovery Compass

The feature predicts how open a listener is to discovering new music during each session and dynamically adjusts recommendation diversity.

Benefits:

- Increased music discovery
- Reduced repetitive listening
- Higher recommendation satisfaction
- Better long-term engagement

# Expected Business Impact

Increase:

- Music Discovery Rate
- Session Duration
- Artist Diversity
- User Retention
- Recommendation Acceptance Rate

Reduce:

- Repeat Listening Ratio
- Recommendation Fatigue

# Conclusion

The AI-powered Review Discovery Engine transforms large volumes of unstructured user feedback into actionable product insights.

These insights support evidence-based product decisions aimed at improving music discovery while maintaining user satisfaction.
